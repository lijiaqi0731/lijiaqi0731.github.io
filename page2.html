UX/UI Redesign

OVERVIEW

After I came to USA, I often use yelp to search for some delicious food. But, sometimes, it looks not very useful. I found there are many disadvantages about Yelp, so I want to choose Yelp to make it better.

DESIGN GOALS

1.The interface is too complex, so I will redesign the organization.

2.The menu of Yelp is not clear, and most of them don’t have picture, which may make user confused, so I think it should normalize the menu.

3.I want to add a function that can order in restaurants. When we go into a restaurant, we open our phone scan quick response code to get the menu of this restaurant. After eating, we can use the money in app to pay.

4.The logo is not very beautiful, thick line and typeface looks not modern.
